# 🌌 Void Terminal

A minimalist, bit-like voice-native AI agent built on the **dTelecom AI Stack**. This terminal provides real-time crypto intelligence and airdrop strategies through an ultra-fast decentralized voice pipeline.

## 🚀 The dTelecom Stack Integration
This project was built for the dTelecom Airdrop Community Program, utilizing the full suite of decentralized voice infrastructure:

* **Real-time Pipeline:** Implements a complete WebRTC → STT → LLM → TTS flow for seamless conversation.
* **Speech-to-Text (STT):** Powered by **Parakeet** for 110x real-time transcription speed, ensuring the agent "hears" instantly.
* **Text-to-Speech (TTS):** Utilizes **Kokoro** for natural, conversational voice synthesis.
* **Autonomous Payments:** Integrated with the **x402 protocol**, allowing the agent to pay for its own infrastructure using **USDC on Base**.
* **Intelligence:** Leverages **Google Gemini 1.5 Flash** as the reasoning engine for concise, expert DeFi insights.

## 🎨 UI/UX: Space-Minimalism
The "Void Terminal" theme features:
* **True Black Aesthetic:** High-contrast, minimal UI to focus on technical pipeline logs.
* **8-Bit Retro Vibes:** Pixelated monospaced typography and CRT-style signal monitors.
* **Live Pipeline Monitoring:** Real-time visual indicators for WebRTC connectivity and STT/TTS status.
